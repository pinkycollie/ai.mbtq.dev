# Super Developer Platform - Backend

This is the FastAPI backend for the Super Developer Platform.

## Replit Setup Instructions

### 1. Import the Project to Replit

1. Go to [Replit](https://replit.com)
2. Click "Create Repl" or "+" button
3. Choose "Import from GitHub"
4. Enter your repository URL
5. Select "Python" as the language
6. Click "Import"

### 2. Configure Environment Variables

Add the following environment variables in the Replit Secrets tab:

- `PORT`: 8080 (Replit uses port 8080 by default)
- `HOST`: 0.0.0.0
- `DATABASE_URL`: Your database connection string (if applicable)

### 3. Run the Application

The `.replit` file is configured to run the application automatically. Just click the "Run" button.

### 4. Access the API

- API Documentation: https://your-repl-name.your-username.repl.co/docs
- API Base URL: https://your-repl-name.your-username.repl.co
